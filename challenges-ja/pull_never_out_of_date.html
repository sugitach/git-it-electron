今回の課題:

コラボレーターが加えた変更をプルすることでファイルを最新に保ちましょう

リモートからプルする

誰かと共同で何か作業を行う場合、コードを最新の状態に保つ必要があります。そのため、行われた変更をプルしたいと思うでしょう。

ステップ: Reporobotは何をしたでしょうか？

Reporobotがあなたの'add-'ブランチに何か変更を加えたかどうか、GitHubの'origin'と呼ばれるリモートからプルすることで確認しましょう。:

$ git pull <REMOTENAME> <BRANCHNAME>

もし何も変更されていなければ、'Already up-to-date'と表示されます。 もし何か変更されていれば、あなたのローカルのバージョンにその変更がマージされます。

Reporobotは変更を加えていましたか？Gitはどこで変更が行われたか教えてくれます。あなたはそのファイルを開き、Reporobotが加えた更新を確認できます。 驚くべきことに、Reporobotは芸術家でした！

Please select your repository directory.

TIPS

  • Gitリポジトリのステータスを確認
    • $ git status
  • リモートブランチの変更をプル
    • $ git pull <REMOTENAME> <REMOTEBRANCH>
  • プルする前にリモートの変更を確認
    • $ git fetch --dry-run
