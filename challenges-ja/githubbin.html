今回の課題:

GitHubアカウントを作成し、Gitの設定にユーザー名を追加する

より良く、いっしょに

ここまでで作成したリポジトリは、あなたのコンピューター上にしかありません。 これは手軽ですが、他の人と共有したり作業したりするのをとても困難にします。 ですが心配はいりません、GitHubはそのためにあるのですから!

ステップ: GitHubアカウントを作成する

GitHubは、そこらじゅうの人々がGit上で取り組んでいるものをアップロード して、簡単にいっしょに作業できるようにするウェブサイトです。

github.comを訪れて、無料アカウントを作成しましょう。 ようこそ!

ステップ: Gitにユーザー名を追加する

GitHubのユーザー名をGitの設定に追加します。 これは、以降の課題を検証するために必要です。GitHubに作成したのとまったく同じ名前を保存します。大文字の部分は大文字で、それから、"<" と ">" の入力は不要なことを思い出してください。

GitHubのユーザー名を設定に追加する:

$ git config --global user.username <USerNamE>

通らない?

GitHubとGit設定のユーザー名が一致しない

よくある間違いは、GitHubのユーザー名と、git configで設定したユーザー名の大文字小文字が一致しないというものです。たとえば、'JLoad'は、'jload'と同じではありません。

Gitのユーザー名を変更するには、先にやったのと同じコマンドを実行するだけです。ただし、大文字小文字は正しくします:

$ git config --global user.username <USerNamE>

更新ができたら、再度 verify してください!
