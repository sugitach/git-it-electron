今回の課題:

Git をあなたのコンピュータにインストールして、あなたの名前とメールアドレスを設定しましょう

Git

Git は Linux の開発者である Linus Torvalds さん開発したオープンソースソフトウェア (誰もが自由に使えるソフトウェア) です。

Git はファイルの変更点を時系列と共に管理できるようにするもので、このようなプログラムはバージョンコントロールプログラムと呼ばれています。

手順: Gitのインストール

  • Windowsの場合:
    GitHub Desktop をダウンロードしてインストールするのがお勧めです。Git 本体も含まれていて、インストールが簡単です。 → desktop.github.com
    Git Shell を端末として使います。
  • MacOSの場合:
    Macの場合でも同様に GitHub Desktop がお勧めです。Git 本体も含まれています。 → desktop.github.com (インストール後にPreferencesの中からコマンドラインツールをインストールすることができます)。
    または、Git 本体のみをダウンロード・インストールすることもできます。 → git-scm.com/downloads 詳細はリンク先のインストール説明を参照してください。

Git はあなたのコンピュータにインストールされている他のプログラムとはちょっと毛色が違います。 デスクトップ上にはアイコンは表示されませんが、コンピュータ内にはちゃんとあって、 いつでも端末(ターミナル)プログラムや GitHub desktop のような Git desktop アプリケーションからアクセスすることができます。

手順: Git の設定

インストールが終ったら (Bash や Shell や Prompt のような)端末(ターミナル) を開いてください。 You can verify that it's really there by typing:

$ git --version

This will return the version of Git that you're running and look something like this:

git version 1.9.1

(Any version 1.7.10 or higher is fine.)

Next, configure Git so it knows who to associate your changes to:

Set your name:

$ git config --global user.name "<Your Name>"

Now set your email:

$ git config --global user.email "<youremail@example.com>"

Tips

Dollar signs $ are often used in programming documentation to signify that the line is command line code (see the code snippets above). You don't actually type the $ in though, only type what comes after. For instance, to run the verify command above, you're actually going to type git-it verify into terminal.
