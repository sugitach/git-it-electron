今回の課題:

Git をあなたのコンピュータにインストールして、あなたのメールアドレスを設定しましょう

Git

Git は Linux オペレーティングシステムを開発した Linus Torvalds の書いたオープンソースソフトウェア (誰もが自由に使えるソフトウェア) です。

このプログラムはファイルの時間の経過にともなう内容の変更を管理できるようにするもので このようなプログラムはバージョンコントロールと呼ばれています。

手順: Gitのインストール

  • Windowsの場合: GitHub Desktopをダウンロードしてインストールするのがお勧めです。 Git が含まれていて、インストールも簡単です: desktop.github.com. Git Shell をターミナルとして使います。
  • Macの場合: Macの場合でも GitHub Desktop がお勧めなのは同じです。Git 含まれています。 desktop.github.com (インストール後にPreferencesの中からコマンドラインツールをインストールすることができます)。または、
    • こちらから Git のみをダウンロードすることができます: git-scm.com/downloads のインストール説明を参照してください。

Git isn't like other programs on your computer. You'll likely not see an icon on your desktop, but it will always be available to you and you'll be able to access it at anytime from your terminal (which you're using in Git-it!) or Git desktop applications (like GitHub Desktop).

Step: Configure Git

Once it is installed, open terminal (aka Bash, aka Shell, aka Prompt). You can verify that it's really there by typing:

$ git --version

This will return the version of Git that you're running and look something like this:

git version 1.9.1

(Any version 1.7.10 or higher is fine.)

Next, configure Git so it knows who to associate your changes to:

Set your name:

$ git config --global user.name "<Your Name>"

Now set your email:

$ git config --global user.email "<youremail@example.com>"

Tips

Dollar signs $ are often used in programming documentation to signify that the line is command line code (see the code snippets above). You don't actually type the $ in though, only type what comes after. For instance, to run the verify command above, you're actually going to type git-it verify into terminal.
