今回の課題:

Git をあなたのコンピュータにインストールして、あなたの名前とメールアドレスを設定しましょう

Git

Git は Linux の開発者である Linus Torvalds さん開発したオープンソースソフトウェア (誰もが自由に使えるソフトウェア) です。

Git はファイルの変更点を時系列と共に管理できるようにするもので、このようなプログラムはバージョンコントロールプログラムと呼ばれています。

手順: Gitのインストール

  • Windowsの場合:
    GitHub Desktop をダウンロードしてインストールするのがお勧めです。Git 本体も含まれていて、インストールが簡単です。 → desktop.github.com
    Git Shell を端末として使います。
  • MacOSの場合:
    Macの場合でも同様に GitHub Desktop がお勧めです。Git 本体も含まれています。 → desktop.github.com (インストール後にPreferencesの中からコマンドラインツールをインストールすることができます)。
    または、Git 本体のみをダウンロード・インストールすることもできます。 → git-scm.com/downloads 詳細はリンク先のインストール説明を参照してください。

Git はあなたのコンピュータにインストールされている他のプログラムとはちょっと毛色が違います。 普通はデスクトップやアプリケーションフォルダにはアイコンが表示されませんが、コンピュータ内にはちゃんとあって、 端末(ターミナル)プログラムや (GitHub desktop のような) Git desktop アプリケーションからアクセスすることができます。

手順: Git の設定

インストールが終ったら 端末(ターミナル) (あるいは Bash, Shell, Prompt)を開いてください。 正しくインストールされていることを確認するために、以下のコマンドを実行してみてください:

$ git --version

うまくいけば次のようなバージョン表記が返ってくるはずです:

git version 1.9.1

(1.7.10以上のバージョンが望ましいです)

次に Git にあなたの名前とメールアドレスを設定して、他の人との区別ができるようにしましょう:

名前の設定:

$ git config --global user.name "<Your Name>"

メールアドレスの設定:

$ git config --global user.email "<youremail@example.com>"

Tips

プログラミング文書では$マークがコマンドラインの目印としてよく使われます (このあとに出てくるコード表記を見てみてください)。 この$マークは入力の必要はなく、それに続けて書かれている文字列のみ入力します。 例えば、git-itの確認コマンドを実行するときには、git-it verify の部分のみをターミナルに入力します。
