Git-it

ようこそ Git-it へ!

Git-it は段階的に進めていくことができるワークショップです。
繁体中国語版や英語版もあります。

このガイドはインストールしたあとはオフラインで動作させることができます。(課題を選択したときの説明を参照してください)。
どうぞ、お楽しみください。

Git-it Challenges

  1. Get Git インストールと Git の設定
  2. Repository ローカルのレポジトリを作成
  3. Commit to it status の確認, 変更の追加(add)と記録(commit)
  4. GitHubbin GitHub アカウントの作成
  5. Remote Control GitHub.com上のリモートレポジトリへ接続
  6. Forks and Clones オープンソースレポジトリのフォークとクローン
  7. Branches aren't just for Birds features/changes のためのブランチを作成
  8. It's a Small World コラボレータ(共同作業者)との共同作業 (Hi, @reporobot!)
  9. Pull, Never Out of Date プッシュ(push)とプル(pull)でGitHub.com と同期
  10. Requesting You Pull Please プルリクエストの作成
  11. Merge Tada ブランチの合成(merge)と削除(delete)
